## Descripción

Portafolio personal moderno con React 19, TailwindCSS 4, Express 5 y PostgreSQL. Optimizado para rendimiento, accesibilidad y SEO.
---

## Tecnologías usadas

### Frontend

- React 19.1.0
- React Router DOM 7.7.1
- TailwindCSS 4.1.11
- GSAP 3.13.0
- Vite 7.0.4
- react-masonry-css
- react-responsive-carousel
- yet-another-react-lightbox
- react-icons
- @fontsource-variable/montserrat
- @fontsource/poppins

### Backend

- Node.js 18+ (ESM)
- Express 5.1.0
- PostgreSQL 15
- Zod 4.0.13
- Pino 9.7.0
- Vitest 3.2.4
- Compression 1.7.5
